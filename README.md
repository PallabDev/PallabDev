## Featured Projects

### 1. Shiftin

A rental management system where tenants can book rental properties and owners can upload and manage houses. This project includes a modern dashboard-driven experience and was built as a hackathon project through ChaiCode.

**Highlights**

- Tenant booking flow and owner property listing workflow
- Modern full stack dashboard experience
- Secure authentication with cookie-based JWT sessions
- Media upload support with Cloudinary
- Production-focused security with Helmet, CORS, and rate limiting
- Email integration with Resend

**Tech Stack**

- Frontend: React 19, Vite 7, React Router 7, Tailwind CSS 4, Axios, Radix UI, Tabler Icons
- Backend: Node.js, Express 5, MongoDB, Mongoose, JWT, Bcrypt, Multer, Cloudinary
- Other: Helmet, CORS, Express Rate Limit, Resend

**Links**

- GitHub: [github.com/PallabDev/shiftin](https://github.com/PallabDev/shiftin)
- Live: [shiftin.luqe.in](https://shiftin.luqe.in)

### 2. BMS POC

A BookMyShow-style hackathon project where users can discover local shows and book seats. The system uses transactional booking logic to prevent the same seat from being reserved by multiple users at the same time.

**Highlights**

- User signup, login, logout, and session refresh
- Email verification and password reset flows
- Movie listing and detailed movie pages
- Seat availability and booking workflow
- My Bookings page for users
- Admin dashboard for movie management
- Cookie-based authentication with CORS credentials support

**Tech Stack**

- Frontend: React, Vite, Tailwind CSS, React Router, Axios
- Backend: Node.js, Express, PostgreSQL, JWT, Joi
- Other: Resend API for verification and password reset emails

**Links**

- Live: [bms.luqe.in](https://bms.luqe.in)
